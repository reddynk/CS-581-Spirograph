# CS-581-Spirograph
Github repo for our CS 581 Comp Fab project to design and print a customizable spirograph set.

## Prerequisites

You will need

* Python 2.7 or above
* [Numpy](http://www.numpy.org)
* [Shapely](https://github.com/Toblerity/Shapely)
